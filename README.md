# 2025-2학기 초급프로젝트 Term Project - JBNU LMS (Clone)

## 1. 프로젝트 개요
React.js와 React Native를 활용하여 **전북대학교 학습관리 시스템(LMS)**의 핵심 기능을 구현한 프로젝트입니다.  
Firebase를 통해 **웹(Web)과 모바일 앱(App) 간의 실시간 데이터 연동**을 구현하였습니다.

* **개발자:** 컴퓨터인공지능학부 성윤오
* **개발 기간:** 2025.10 ~ 2025.12

## 2. 폴더 구조
이 저장소는 두 개의 프로젝트로 구성되어 있습니다.
* **📂 attendance-web/** : 교수/학생용 웹 사이트 (React.js)
* **📂 attendance-app/** : 학생용 모바일 애플리케이션 (React Native / Expo)

## 3. 주요 기능
* **공통:** Firebase Authentication (로그인), Firestore Database (데이터 연동)
* **Web:** 강의 자료 업로드, 출석 번호 생성, 과제 관리
* **App:** 실시간 출석 체크, 나의 출석 현황 확인
